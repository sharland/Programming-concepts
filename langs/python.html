What is programming?

When you woke up this morning many of the things you used or did would have relied on instructions being performed by a computer.

The coffee machine you or your parents would have used using basic instructions to control how the machine functions

The electronic bus timetable at the bus-stop will be programmed to fetch information on bus arrivals.

A smart TV or home automation device might be programmed to fetch the latest information about the weather from the web.

All the instructions on these devices will have been written by a human to make sure the machine worked as expected.

This is programming! It is the process of writing sets of instructions in order to get a computer to perform a specific task.

Let's now move onto finding out how to actually create simple instructions on a computer that will lead to the computer doing something for you!

Think of the control! Think of the power!

Values

The answers you would have given can be described as values

These values could change. If you came by bus to school today, tomorrow you could come by car. The temperature is always changing and although Ariana Grande could be your favourite singer now, it could be someone different a year from now.

Each of these values can also be linked with a label which describes what that value is.

For example:

  1. Method of Transport (label) holds bus (value)
  2. tempToday (label) holds 19 (value)
  3. faveSinger = "Ariana Grande"

Activity 1: Writing your first code

In the Python Shell enter each line as shown below pressing enter afterwards


name = "Bob"
age = 135
print("Wow {0} you are {1} years old".format(name,age))

Understanding the code from activity 1

  1. The variable "name" holds the value Bob
  2. The variable "age" holds the value 135
  3. The print function prints a string of text and then uses the format method to provide the two variables name and age which are then placed into the corresponding format codes {0} and {1} in the code.

As you can see from the code that the two variables which we setup, name and age, can then be used again in the print statement instead of having to retype their values.
